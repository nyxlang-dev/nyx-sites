Un lenguaje para software nativo y robusto.

Nyx compila a código nativo a través de LLVM, hospeda su propio compilador y habla un dialecto legible por máquina que los agentes de IA pueden escribir, leer y corregir. Confiable por diseño, no por convención.

empezar $ curl -sSf nyxlang.com/install.sh | sh
$ nyx run src/main.nx
src/main.nx ─────────────────────────────┐
// enums con datos · pattern matching exhaustivo
// compilado a un binario nativo vía LLVM.

enum Shape {
    Rect(int, int),
    Circle(int)
}

fn area(s: Shape) -> int {
    return match s {
        Shape.Rect(w, h) => w * h,
        Shape.Circle(r)  => 3 * r * r
    }
}

fn main() {
    let shapes: Array<Shape> = [
        Shape.Rect(3, 4),
        Shape.Circle(5),
        Shape.Rect(6, 2)
    ]
    var total: int = 0
    var i: int = 0
    while i < shapes.length() {
        total = total + area(shapes[i])
        i = i + 1
    }
    print("figuras:    " + int_to_string(shapes.length()))
    print("área total: " + int_to_string(total))
}

Autohospedado y rápido

El compilador está escrito en Nyx y se compila a sí mismo — recompilar da IR byte-idéntico, verificado en cada release. La salida es código nativo vía LLVM, a la par o por encima de C en cómputo.

[ai] ✦ ai-first

Legible para agentes

Diagnósticos legibles por máquina (NYX_DIAG=json, códigos estables, línea:columna, sugerencias de fix), AGENTS.md / CAPABILITIES.md autogenerados, e inferencia en el dispositivo vía std/llm.

[eco] ✦ completo

Un ecosistema entero

Un framework web, un store compatible con Redis, un proxy inverso, un editor de terminal — todo escrito en Nyx. Todo lo que sirve este sitio es uno de ellos.
