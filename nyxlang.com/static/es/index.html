Un lenguaje para software robusto y nativo.

Nyx compila a código nativo a través de LLVM, hospeda su propio compilador y habla un dialecto machine-readable que los agentes de IA pueden escribir, leer y arreglar. Confiable por diseño — no por convención.

// Una muestra de Nyx — enums con datos, pattern matching exhaustivo,
// compilado a un binario nativo vía LLVM.

enum Shape {
    Rect(int, int),
    Circle(int)
}

fn area(s: Shape) -> int {
    return match s {
        Shape.Rect(w, h) => w * h,
        Shape.Circle(r)  => 3 * r * r
    }
}

fn main() {
    let shapes: Array<Shape> = [
        Shape.Rect(3, 4),
        Shape.Circle(5),
        Shape.Rect(6, 2)
    ]

    var total: int = 0
    var i: int = 0
    while i < shapes.length() {
        total = total + area(shapes[i])
        i = i + 1
    }

    print("figuras:    " + int_to_string(shapes.length()))
    print("área total: " + int_to_string(total))
}

Por qué Nyx

Tres cosas que hace bien

✦ nativo

Self-hosting y rápido

El compilador está escrito en Nyx y se compila a sí mismo — recompilar produce IR byte-idéntico, verificado en cada release. La salida es código nativo vía LLVM, a la par o mejor que C en cómputo.

✦ ai-first

Legible para agentes

Diagnósticos machine-readable (NYX_DIAG=json, códigos estables, line:column, sugerencias de arreglo), AGENTS.md / CAPABILITIES.md autogenerados, e inferencia LLM on-device vía std/llm.

✦ completo

Un ecosistema entero

Un framework web, un almacén compatible con Redis, un reverse proxy, un editor de terminal — todos escritos en Nyx. Todo lo que sirve este sitio es uno de ellos.

AI-first, en concreto

Errores sobre los que un modelo puede actuar

Cada diagnóstico es una línea de JSON con un código estable, ubicación exacta y una sugerencia de arreglo — en las fases de parse y semantic, bilingüe. Sin scrapear stderr, sin adivinar.

Constrúyelo en Nyx

Un lenguaje, un stack completo

Cada uno de estos es un producto separado, escrito enteramente en Nyx, corriendo en producción detrás de este dominio.

nyx-serve
Framework web — rutas, JSON, WebSockets, plantillas, multipart, sesiones. Un binario estático.
nyx-kv
Almacén clave-valor compatible con Redis — 64 comandos, Pub/Sub, SCAN estable ante resize, TLS, multi-tenant.
nyx-proxy
Reverse proxy HTTPS — TLS con SNI, health checks, cache LRU, Prometheus /metrics, rate limiting.
nyx-edit
Editor de texto para terminal en un archivo Nyx — resize en vivo, UTF-8, mouse, undo de 1000 niveles, sin parpadeo.

Empezar

De cero a ejecutar en 30 segundos

Instala el toolchain, crea un proyecto, ejecútalo. Sin runtime que instalar en tus servidores — el compilador produce un binario estático.

$ curl -sSf https://nyxlang.com/install.sh | sh

Linux (x86_64, ARM64) · requiere Clang · verificar: nyx --version → nyx 0.22.0

$ nyx init mi-app
$ cd mi-app && nyx run
