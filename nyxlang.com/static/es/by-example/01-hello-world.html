Hello World

Todo camino en programación empieza acá. Este es el programa Nyx completo más pequeño posible: define una función main, llama a print dos veces y devuelve 0 para indicar éxito. Sin imports, sin código de relleno — solo el código que importa.

Código

// Programa mínimo: imprime un saludo al mundo

fn main() -> int {
    print("Hello, world!")
    print("Bienvenido a Nyx")
    return 0
}

Salida

Hello, world!
Bienvenido a Nyx

Explicación

Todo programa Nyx comienza su ejecución desde la función main. El tipo de retorno -> int le indica al compilador que main devuelve un entero — por convención, 0 significa éxito y cualquier otro valor le señala un error al sistema operativo.

La función incorporada print escribe una cadena en la salida estándar seguida de un salto de línea. Acepta un único argumento de tipo String. A diferencia de muchos lenguajes, Nyx no requiere importar nada para usar print — siempre está disponible.

Los comentarios empiezan con // y se extienden hasta el final de la línea. El compilador los ignora y existen solo para quienes leen el código.
