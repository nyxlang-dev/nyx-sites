1 · Instalar el toolchain

nyx-serve es una librería para el lenguaje Nyx, así que lo único que instalás es el toolchain de Nyx — el compilador, el gestor de paquetes y la librería estándar, en un directorio bajo tu home. 

$ curl -sSf https://nyxlang.com/install.sh | sh

El instalador pone todo en ~/.nyx y enlaza el comando nyx en ~/.local/bin. Si ese directorio no está en tu PATH, el instalador te dice la línea exacta para agregar a tu perfil de shell.

Verificar

$ nyx --version
nyx 0.22.0

Nyx compila a través de LLVM, así que también necesitás clang — la mayoría de las distros Linux y macOS ya lo traen. Verificá con clang --version; en Debian/Ubuntu es apt install clang.

Nada más. No hay runtime que instalar en tus servidores — el compilador produce un binario estático que se lleva todo el framework adentro.
